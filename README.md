# checkSelf-auto
코로나 자가진단 자동화 스크립트

# 사용법
1. 노드 패키지 설치
2. `cd /경로`
<br/>예시 : `cd C:\Users\aaa\Desktop\checkSelf`
3. `npm start`
4. http://localhost:3000/storage?name=이름&birth=생일
<br/>예시 : http://localhost:3000/storage?name=지성빈&birth=021202
5. http://localhost:3000/push
